# Domain list community

This project manages a list of domains, to be used as geosites for routing purpose in [SagerNet/sing-box](https://github.com/SagerNet/sing-box).

Based on [elysias123/geosite](https://github.com/elysias123/geosite)

You could download the lastest `geosite.db` from [https://github.com/zuohl/sing-geosite/releases/latest/download/geosite.db](https://github.com/zuohl/sing-geosite/releases/latest/download/geosite.db) or [https://cdn.jsdelivr.net/gh/zuohl/sing-geosite@release/geosite.db](https://cdn.jsdelivr.net/gh/zuohl/sing-geosite@release/geosite.db) with CDN powered by jsdelivr.
